## Como executar o projeto

```bash
# Clonar repositório
git clone https://github.com/williamtrevisan/ddd-patterns.git
git clone git@github.com:williamtrevisan/ddd-patterns.git

# Entrar na pasta do projeto
cd ddd-patterns

# Instalar dependências
npm install

# Rodar os testes
npm test
```
